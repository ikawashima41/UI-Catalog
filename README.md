## How to setup

### Environment

- Swift ver4.2
- Xcode ver10.1 & Command line tools for Xcode ver10.1 
- Carthage ver0.32.0
- Swiftlint Latest version

### Libruary Setup

プロジェクトをcloneしてきたあとにスクリプト実行

```
$ sh bootstrap.sh
```

### CI/CD

- Bitrise
   - Trigger(push develop branch/pull request to develop)
- Fastlane
- Danger
